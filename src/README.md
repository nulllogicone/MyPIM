# MyPIM

Blazer-based Personal Information Manager (PIM) application.
## Overview
MyPIM is a web application built with Blazor that helps users manage their personal information efficiently. It provides features such as request tracking, data management, and integration with external services.
## Features
- User authentication and authorization
- Request submission and tracking   
- Data management with CRUD operations
- Integration with Microsoft Graph API  
- Responsive UI design
